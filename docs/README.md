# StudentBank SaaS Project

## Overview
This is the StudentBank SaaS project, built with a Node.js + Express backend and a React + Vite frontend.  

## Structure
- **backend/**: Contains the server-side code using Node.js, Express, Prisma, and PostgreSQL.
- **frontend/**: Contains the client-side code using React and Vite.
- **docs/**: Documentation for project setup and usage.
